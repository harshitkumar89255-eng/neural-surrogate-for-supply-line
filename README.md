# Neural Surrogate Supply Chain Simulator

This project implements a neural surrogate model for supply chain simulation and optimization.

## Features

- Supply chain simulation engine
- Neural surrogate modeling for fast predictions
- Disruption modeling
- Visualization tools

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the simulator: `python main.py`

## Usage

Configure the simulation in `configs/base_config.py` and run `main.py`.

## Project Structure

- `simulator/`: Core simulation modules
- `configs/`: Configuration files
- `visualization/`: Graph visualization tools
- `data/`: Data storage
- `main.py`: Entry point
- `requirements.txt`: Dependencies
- `README.md`: This file
